##### 方案 A: 字体加载策略优化

**位置**: `src/app/layout.tsx`

**Before**:
```typescript
const instrumentSerif = Instrument_Serif({
  variable: "--font-display",
  subsets: ["latin"],
  weight: ["400"],
  display: "swap",  // ❌ 会导致 CLS
  preload: true,
});
```

**After**:
```typescript
const instrumentSerif = Instrument_Serif({
  variable: "--font-display",
  subsets: ["latin"],
  weight: ["400"],
  display: "optional",  // ✅ 不会交换，避免 CLS
  preload: true,
});
```

**原理**:
- `display: swap`: 先显示fallback字体，加载完成后交换 → CLS
- `display: optional`: 如果100ms内未加载完，本次页面加载使用fallback，下次再用 → 无 CLS

**适用字体**:
- Instrument Serif (标题字体)
- DM Sans (正文字体)
- JetBrains Mono (代码字体)

### Font Loading Strategy 对比

| Strategy | 工作原理 | CLS 影响 | 适用场景 |
|----------|----------|----------|----------|
| `auto` | 浏览器决定 | 中 | 默认值，不推荐 |
| **`block`** | 3秒内阻塞，直到字体加载 | **低** | 品牌字体必须显示 |
| `swap` | 立即显示fallback，加载后交换 | **高** | 性能优先，可接受闪烁 |
| **`fallback`** | 100ms block，3秒swap窗口 | **中** | 平衡方案 |
| **`optional`** ⭐ | 100ms内加载则用，否则等下次 | **零** | 性能优先，字体可选 |

**我们的选择**: `optional`
- ✅ 完全避免 font-swap CLS
- ✅ 首次访问：使用 fallback (Georgia/DM Sans)
- ✅ 二次访问：字体已缓存，立即使用
- ⚠️ 首次访问字体可能不显示（可接受）

**Alternative**: `fallback`
- 100ms block + 3s swap 窗口
- 更激进的字体加载策略
- 仍然有小概率 CLS

## Best Practices 总结

### 1. 防止 CLS 的通用策略

**Always**:
- ✅ 为所有图片/视频指定 `width` 和 `height`
- ✅ 预留空间给动态内容（`min-height`, `aspect-ratio`）
- ✅ 使用 `font-display: optional` 或 `fallback`
- ✅ 避免在首屏内容上方插入内容
- ✅ 使用 `transform` 和 `opacity` 做动画（不用 `top/left`）

**Never**:
- ❌ 使用 `font-display: swap` (除非可接受 CLS)
- ❌ 在页面加载期间动态插入 `position: fixed/absolute` 元素
- ❌ 使用无尺寸的图片/iframe
- ❌ 在现有内容上方插入广告/通知

### 2. 固定定位叠加层的最佳实践

**如果必须使用 `position: fixed` 叠加层**:
1. **延迟加载**: 在 LCP 后再显示（如本次的 2500ms 延迟）
2. **预渲染**: 始终在 DOM 中，用 `opacity: 0` 隐藏
3. **CSS 优化**: 
   - `will-change: transform/opacity`
   - `contain: layout style paint`
   - 仅使用 `transform` 和 `opacity` 动画
4. **用户触发**: 基于用户交互而非自动显示

### 3. Font Loading 最佳实践

**推荐策略**:
```typescript
// 关键品牌字体 - 使用 block
const brandFont = Font({
  display: "block",
  preload: true,
})

// 正文字体 - 使用 optional
const bodyFont = Font({
  display: "optional",
  preload: true,
})

// 代码字体 - 使用 optional
const monoFont = Font({
  display: "optional",
  preload: false,  // 非首屏，无需 preload
})
```

**Fallback Fonts 选择**:
- Serif: `Georgia, 'Times New Roman', serif`
- Sans-serif: `system-ui, -apple-system, BlinkMacSystemFont, 'Segoe UI', sans-serif`
- Monospace: `'Courier New', monospace`
