### ✅ Task #173: 修复 Accessibility - 添加 Main Landmark

**问题**:
- Lighthouse 审计发现："Document does not have a main landmark."
- 影响屏幕阅读器用户导航
- 不符合 WCAG 2.1 AA 标准

**解决方案**:

#### 修改 src/app/page.tsx

**Before**:
```tsx
export default function HomePage() {
  return <WorkspaceContainer />
}
```

**After**:
```tsx
export default function HomePage() {
  return (
    <main role="main" className="h-screen">
      <WorkspaceContainer />
    </main>
  )
}
```

**修改要点**:
- 添加 `<main>` 标签包裹主要内容
- 显式声明 `role="main"` (为了兼容性)
- 保持 `h-screen` 类名，确保全屏布局

**预期改进**:
- Accessibility 评分从 98/100 提升到接近 100/100
- 符合语义化 HTML 最佳实践
- 屏幕阅读器用户可以快速跳转到主要内容
