## 快速开始

### 首次使用引导

打开应用后，你会看到 **Welcome** 欢迎页面。系统会自动显示首次使用引导，带你了解核心功能。

**首次引导包含三个阶段**：
1. **Welcome 页面**：了解如何描述创意和选择模板
2. **Chat 对话**：学习如何与 AI 对话生成脚本和分镜
3. **Timeline 编辑**：掌握编辑、重排和导出视频的方法

> 💡 **提示**：如果你跳过了引导，可以点击右下角的 <kbd>Shift + ?</kbd> 查看所有快捷键和功能说明。

### 三种开始方式

#### 方式 1：描述创意（推荐）

1. 在搜索框中输入你的视频创意，例如：
   - "猫咪日常" — 拍摄宠物的日常生活
   - "产品宣传片" — 介绍你的产品特点
   - "旅行 Vlog" — 记录旅行见闻
   - "教程视频" — 制作教学内容

2. 点击 **开始创作**，AI 会和你对话，询问：
   - 视频时长（10秒 ~ 5分钟）
   - 画面比例（16:9 横屏 / 9:16 竖屏）
   - 风格偏好（写实/卡通/动漫等）

3. 回答 AI 的问题后，系统会自动生成：
   - 📝 **脚本**：完整的文案和场景描述
   - 🎬 **分镜**：每个场景的视觉画面
   - 🖼️ **图片**：AI 生成的配图

#### 方式 2：使用预设模板

点击 **或从预设模板开始**，选择一个模板：

| 模板 | 适用场景 | 时长 | 比例 |
|------|---------|------|------|
| 商业广告 | 产品宣传、品牌推广 | 30秒 | 16:9 |
| 教程视频 | 知识讲解、技能教学 | 2-3分钟 | 16:9 |
| Vlog | 生活记录、旅行分享 | 1-2分钟 | 9:16 |
| 音乐MV | 歌曲配乐、创意视频 | 1分钟 | 16:9 |
| 短剧 | 剧情叙事、情感故事 | 3-5分钟 | 9:16 |
| TikTok | 短视频创作、快节奏内容 | 15-30秒 | 9:16 |
| Instagram | 社交媒体内容 | 30-60秒 | 1:1 |

选择模板后，系统会自动填充默认内容，你可以直接编辑。

#### 方式 3：从历史项目继续

如果你之前创建过项目，在左侧边栏的 **项目列表** 中点击项目名称，即可继续编辑。

## 核心功能

### 1. 对话式脚本生成

**AI 会理解你的创意，自动生成脚本**

**对话流程**：
1. AI 询问视频时长 → 你回答"30秒"
2. AI 询问画面比例 → 你选择"9:16 竖屏"
3. AI 询问风格偏好 → 你选择"写实摄影"
4. AI 自动生成完整脚本和分镜

**生成内容包括**：
- 📝 **场景描述**：每个镜头的文案和动作
- 🎨 **视觉提示**：画面构图和风格建议
- ⏱️ **时长分配**：每个场景的秒数
- 🎬 **镜头运动**：推拉摇移等镜头语言

> 💡 **提示**：生成过程中，你可以随时修改需求，AI 会重新生成。

---

### 2. 分镜编辑（Timeline Editor）

生成分镜后，你会进入 **Timeline 编辑器**，这是最强大的功能模块。

#### 界面布局

```
┌─────────────────────────────────────────────────────────────┐
│  [场景列表]     [预览区域]                    [快捷操作]    │
│   ├─ 场景1     ┌──────────────────┐          ┌──────────┐  │
│   ├─ 场景2     │                  │          │ 添加场景 │  │
│   ├─ 场景3     │   选中场景预览   │          │ 批量删除 │  │
│   └─ ...       │                  │          │ 导出视频 │  │
│                └──────────────────┘          └──────────┘  │
│────────────────────────────────────────────────────────────│
│  [时间轴]  ▶ [场景1] [场景2] [场景3] [场景4] ...          │
└─────────────────────────────────────────────────────────────┘
```

#### 基本操作

**选择场景**：
- 点击左侧列表中的场景，右侧会显示大图预览
- 点击底部时间轴中的场景，快速跳转

**编辑场景**：
- **描述**：点击场景描述文字，直接修改
- **时长**：点击时长数字，输入新的秒数（1-10秒）
- **删除**：点击删除按钮，或按 <kbd>Delete</kbd>

**重排场景**：
- 在左侧列表中拖拽场景上下移动
- 在底部时间轴中拖拽场景左右移动

**复制场景**：
- 选中场景，按 <kbd>Cmd/Ctrl + D</kbd>
- 或点击场景菜单中的"复制"按钮

#### 批量操作

**多选场景**：
- <kbd>Cmd/Ctrl + 点击</kbd>：选择多个场景（不连续）
- <kbd>Shift + 点击</kbd>：选择连续的场景
- <kbd>Cmd/Ctrl + A</kbd>：全选所有场景

**批量删除**：
- 选中多个场景后，点击"批量删除"按钮
- 或按 <kbd>Delete</kbd> 键

> ⚠️ **注意**：批量删除会自动保存快照，可以通过版本历史恢复。

---

### 3. 视频导出

编辑完成后，点击右上角的 **导出视频** 按钮。

#### 导出配置

| 选项 | 说明 | 推荐值 |
|------|------|--------|
| **分辨率** | 视频清晰度 | 1080p（默认） |
| | 720p — 适合网络传输 | |
| | 1080p — 高清画质 | |
| | 4K — 超高清（慢） | |
| **帧率** | 画面流畅度 | 30 FPS（默认） |
| | 24 FPS — 电影感 | |
| | 30 FPS — 通用标准 | |
| | 60 FPS — 超流畅 | |
| **音频** | 背景音乐 | 可选（预设或上传） |
| **字幕** | 自动字幕 | 可选（ASR生成） |

#### 导出流程

1. 选择配置后，点击 **开始导出**
2. 系统会显示实时进度：
   - ✅ 渲染场景 1/5
   - ✅ 渲染场景 2/5
   - ⏳ 合成视频...
   - ✅ 导出完成！

3. 导出完成后，点击 **下载视频** 保存到本地

> 💡 **提示**：导出时长约为视频时长的 2-3 倍。例如，30秒视频需要 1-2 分钟。

## 进阶功能

### 1. 音频配乐

在 **Export Panel** 中，你可以为视频添加背景音乐。

#### 使用预设音乐

点击 **选择音乐** → 从 8 首免费音乐中选择：
- 🎵 轻快欢乐
- 🎵 温馨浪漫
- 🎵 激情活力
- 🎵 科技未来
- ...

#### 上传自定义音乐

点击 **上传音频** → 选择本地音频文件（支持 MP3、WAV、AAC）

#### 音量控制

拖动音量滑块，调整背景音乐音量（0-100%）

> 💡 **提示**：音乐会自动循环或裁剪以匹配视频时长。

---

### 2. 字幕系统

#### 自动生成字幕

1. 在 Timeline 编辑器中，点击 **生成字幕** 按钮
2. 上传音频文件（如果有配音）
3. 系统会自动识别语音，生成字幕时间轴
4. 字幕会显示在视频底部

#### 手动编辑字幕

点击字幕轨道中的字幕条，可以：
- 修改文字内容
- 调整显示时间
- 删除或添加字幕

#### 导出字幕

点击 **导出 SRT** 按钮，保存字幕文件，可用于其他视频编辑软件。

---

### 3. 版本历史和撤销

#### 自动快照

系统会在以下操作时自动保存快照：
- 删除场景
- 重排场景
- 批量操作

#### 查看历史版本

点击左上角的 **版本历史** 按钮，查看最近 10 个版本。

#### 恢复版本

点击版本列表中的 **恢复** 按钮，回退到该版本。

> ⚠️ **注意**：恢复版本会覆盖当前编辑内容，请谨慎操作。

---

### 4. 多项目管理

#### 创建新项目

点击左上角的 **新建项目** 按钮，开始一个全新的视频创作。

#### 切换项目

在左侧边栏的项目列表中，点击项目名称，快速切换。

#### 重命名项目

右键点击项目名称 → 选择 **重命名**，输入新名称。

#### 删除项目

右键点击项目名称 → 选择 **删除**。

> ⚠️ **注意**：删除操作无法撤销！请确保已导出视频后再删除。

## 最佳实践

### 1. 脚本生成技巧

**明确你的需求**：
- ❌ "做个视频" → 太模糊
- ✅ "30秒的产品宣传片，介绍我们的智能手表，风格要科技感" → 清晰明确

**分步描述**：
- ❌ 一次性描述所有细节
- ✅ 先说主题，AI 会引导你逐步完善

**参考示例**：
- "像 Apple 风格的产品展示"
- "模仿旅行 Vlog 的节奏感"
- "参考《流浪地球》的科幻风格"

---

### 2. 分镜编辑技巧

**时长分配原则**：
- **重要镜头**: 5-7 秒（突出重点）
- **过渡镜头**: 2-3 秒（节奏流畅）
- **快节奏镜头**: 1-2 秒（增加动感）

**场景顺序**：
- 开场：吸引注意（3-5秒）
- 主体：展开内容（60-70%时长）
- 结尾：行动号召（5-10秒）

**画面构图**：
- 横屏（16:9）：适合风景、产品展示
- 竖屏（9:16）：适合人物特写、手机观看
- 方形（1:1）：适合社交媒体（Instagram）

---

### 3. 导出优化建议

**分辨率选择**：
- **720p**：适合网络传输，文件小（推荐移动端）
- **1080p**：高清画质，平衡质量和大小（推荐通用）
- **4K**：超高清，适合专业用途（渲染慢，文件大）

**帧率选择**：
- **24 FPS**：电影感，柔和
- **30 FPS**：通用标准，适合大部分场景
- **60 FPS**：超流畅，适合运动画面

**音频建议**：
- 背景音乐音量：建议 30-50%（不要盖过画面）
- 避免使用有版权的音乐
- 上传音频时，确保格式正确（MP3/WAV）

## 故障排查

### 1. 生成失败

**问题**：点击"开始创作"后，长时间无响应

**可能原因**：
- 网络连接中断
- API 限流（请求过于频繁）
- 服务器故障

**解决方法**：
1. 检查网络连接
2. 等待 30 秒后重试
3. 刷新页面，从历史项目继续
4. 如果持续失败，联系技术支持

---

### 2. 图片加载失败

**问题**：分镜中的图片显示为占位符或空白

**可能原因**：
- 图片生成服务故障
- 网络延迟
- 浏览器缓存问题

**解决方法**：
1. 刷新页面
2. 清空浏览器缓存
3. 重新生成该场景的图片

---

### 3. 导出卡住

**问题**：导出进度长时间停留在某个百分比

**可能原因**：
- 视频时长过长（>5分钟）
- 分辨率太高（4K）
- 服务器负载过高

**解决方法**：
1. 等待 5 分钟（大文件需要时间）
2. 降低分辨率（改为 1080p 或 720p）
3. 分段导出（将长视频拆分为多个短视频）

---

### 4. 快捷键不生效

**问题**：按下快捷键没有反应

**可能原因**：
- 焦点在输入框中（快捷键被禁用）
- 浏览器快捷键冲突
- 操作系统快捷键冲突

**解决方法**：
1. 确保没有在输入框中
2. 使用鼠标点击空白区域，获取焦点
3. 查看 <kbd>Shift + ?</kbd> 帮助面板，确认快捷键正确
