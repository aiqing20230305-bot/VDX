## 🎯 优化目标

基于 `CHAT_UX_IMPROVEMENTS_2026-04-11.md` 中的"短期建议"，统一系统指导消息格式，提升识别度。

---

## ✨ 实施内容

### 统一前缀格式

为所有系统指导类消息添加 **"💡 提示："** 前缀，形成统一的视觉和语言风格。

#### 修改文件

**1. src/lib/ai/chat-agent.ts**
```diff
- content: `👋 我是**超级视频Agent**，你的AI视频制作伙伴。

- 我可以帮你：
- - 🎯 **从选题到成片**：描述想法 → 脚本 → 分镜 → 视频
- - 🖼️ **图片生视频**：上传图片 → 自动创作
- - 🎬 **视频二创**：上传视频 → 分析 → 改造重生
- - 💬 **聊着聊着就出来了**：告诉我你想做什么

- 你想从哪里开始？`,

+ content: `💡 提示：欢迎使用**超级视频Agent**

+ 我可以帮你完成以下创作：
+ • 🎯 **选题生成视频** — 告诉我想法，自动生成脚本、分镜、视频
+ • 🖼️ **图片生成视频** — 上传产品图/场景图，自动创作
+ • 🎬 **视频二创改造** — 上传视频，AI分析并重新创作

+ **👇 请选择一个起点**，或直接描述你的需求：`,
```

**改进点**:
- ✅ 添加统一的"💡 提示："前缀
- ✅ 文案更简洁（去除"聊着聊着就出来了"等口语化表述）
- ✅ 使用 "•" 而非 "-"，视觉更清晰
- ✅ 明确指令："**👇 请选择一个起点**"

---

**2. src/lib/ai/chat-actions.ts**

同步修改客户端版本的欢迎消息（与 chat-agent.ts 保持一致）。


**改进点**:
- ✅ 统一使用"💡 提示："前缀
- ✅ 去除多余的冒号和"我"字，更简洁

## 🎨 视觉对比

### Before
```
👋 我是超级视频Agent，你的AI视频制作伙伴。
我可以帮你：...

✅ 请确认上述创意方案...

💡 我为你推荐了以下选题...
```

**问题**: 前缀不统一（👋 / ✅ / 💡），识别度不一致

### After
```
💡 提示：欢迎使用超级视频Agent
我可以帮你完成以下创作：...

💡 提示：请确认上述创意方案...

💡 提示：已为你推荐以下选题...
```

**优势**: 统一的"💡 提示："前缀，与视觉增强（6px左边框+微光）形成完整体验
