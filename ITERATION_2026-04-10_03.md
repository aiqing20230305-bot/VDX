## 完成任务

### Task #168: 实现精确的workflow步骤追踪 ✅

**目标**: 从粗粒度3步升级为细粒度4步进度显示

**背景**:
- 之前的 WorkflowProgress 只能区分3个大阶段（chat/timeline/export）
- Chat 阶段内部有多个子步骤（选题→脚本→分镜），但用户看不到
- 缺乏透明度，用户不清楚AI在做什么

**实现方案**:

#### 1. ChatPanel 改进
```typescript
// 导出类型供父组件使用
export type FlowStage = 'understanding' | 'scripting' | 'storyboarding' | 'completing' | null

// 添加回调prop
interface ChatPanelProps {
  ...
  onFlowStageChange?: (stage: FlowStage) => void
}

// 统一状态更新函数
const updateFlowStage = (stage: FlowStage) => {
  setFlowStage(stage)
  onFlowStageChange?.(stage)
}
```

#### 2. WorkspaceContainer 改进
```typescript
// 接收 flowStage
const [flowStage, setFlowStage] = useState<FlowStage>(null)

// 根据 flowStage 计算精确步骤
const chatStep = flowStage === 'understanding' ? 0
  : flowStage === 'scripting' ? 1
  : flowStage === 'storyboarding' ? 2
  : 0
```

**效果**:
- ✅ 用户可以实时看到当前处于哪个子步骤
- ✅ 流程透明度大幅提升
- ✅ 进度条准确反映AI处理阶段

**文件修改**:
- `src/components/workspace/ChatPanel.tsx` - 添加flowStage回调机制
- `src/components/workspace/WorkspaceContainer.tsx` - 精确步骤计算
