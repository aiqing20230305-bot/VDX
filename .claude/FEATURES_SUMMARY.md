# 超级视频Agent - 功能总览（v1.0）

## 🎯 核心创作闭环

```
选题/图片 → 脚本生成 → 分镜图 → 视频生成 → 关键帧提取 → 作为参考 → 保持一致性 ✨
```

---

## ✨ 已实现功能

### 1. 📝 智能脚本生成
- 基于选题生成多个创意方案
- 支持图片上传自动创作
- 视频二创分析与改造
- 交互式参数选择（时长/比例/风格）

### 2. 🎬 分镜系统

#### 2.1 分镜生成
- Claude 深度理解脚本生成精准提示词
- 5种视觉风格（真实/电影/动漫/3D/插画）
- **简化风格预设**：移除过度技术细节，保持简洁高效
- 支持文生图 + 图生图两种模式
- 实时进度展示（GenerationProgress）
- **智能提示词优化**：
  - 自动控制提示词长度（50-80词）
  - 移除冗余技术参数
  - 符合平台规范
  - 提高生成成功率

#### 2.2 参考图智能处理
- **自动分类**：人物/产品/场景
- **产品一致性引擎**：
  - 深度分析产品特征（摄像头/Logo/材质）
  - 生成约束提示词防止出错
  - **不描述颜色**（由参考图决定）
  - 高风险产品多视图建议
- **人物风格转换**：真人照片 → CG/动漫风格

#### 2.3 分镜编辑
- **对话式修改**：
  - 自然语言修改提示词
  - 支持单帧/多帧/全局修改
  - 6大修改维度（光线/色调/镜头/情绪/细节/构图）
- **批量选择重新生成**：
  - Checkbox 多选模式
  - 只重新生成不满意的帧
  - 节省时间和token
- **单帧精细调整**：
  - 点击查看详情
  - 独立重新生成

#### 2.4 分镜展示
- 网格展示（9:16竖屏/16:9横屏自适应）
- **合成概览图**：一键生成带序号的网格图
- 实时计数（已选帧数/总时长）

### 3. 🎥 视频生成

#### 3.1 帧选择生成
- **选择想用的帧**：勾选满意的帧生成
- **跳过不好的帧**：提升最终质量
- 实时统计（已选帧数/总时长）
- 引擎选择（Seedance/可灵）

#### 3.2 多引擎支持
- 即梦 Seedance 2.0（高质量）
- 可灵 AI（快速）
- 智能调度与API集成

#### 3.3 关键帧提取 ⭐
- **从视频提取关键帧**：
  - 自动均匀提取（N帧）
  - 场景切换检测（智能）
  - 间隔提取（每N秒）
  - 特定时间点提取
- **用于后续参考**：
  - 保持角色外观一致
  - 延续场景风格
  - 故事类视频必备
- **闭环创作**：
  ```
  生成视频 → 提取帧 → 作为参考图 → 生成新分镜 → 角色一致性 ✨
  ```

### 4. 🛡️ 违禁词智能绕过 + 提示词简化

#### 4.1 自动检测与替换
- **品牌侵权**：iPhone → smartphone, Nike → athletic shoes
- **暴力血腥**：blood → red liquid, gun → weapon, 血 → 受伤
- **负面情绪**：suicide → sadness, 自杀 → 悲伤
- **成人内容**：nude → figure, sexy → attractive
- 支持中英文违禁词

#### 4.2 智能替换策略
- 同义词替换（保留语义）
- 泛化描述（品牌 → 通用）
- 温和表达（暴力 → 中性）

#### 4.3 提示词简化 ⭐
- **自动简化**：生成后自动简化提示词
- **移除冗余**：删除重复的技术参数和形容词
- **长度控制**：保持在200字符以内
- **符合规范**：确保平台合规性
- **核心保留**：主体、场景、光线、动作

#### 4.4 集成到生成流程
- 分镜提示词自动过滤
- 自动简化确保合规
- 视频生成前检测
- 实时反馈替换结果

### 5. 💬 上下文理解系统

#### 5.1 智能意图识别
- 自动理解用户回答是在回应什么
- 上下文状态管理（waiting_image_confirmation等）
- 对话历史增强（15条，含action）

#### 5.2 图片上传对齐流程
```
上传图片 → AI分析 → 展示结果 → 用户确认/修正 → 推演脚本 ✅
```
- 不直接生成，先对齐理解
- 给用户修正机会
- 透明可控

### 6. 🎨 用户体验优化

#### 6.1 生成进度
- 实时进度条
- 阶段动效（emoji + 描述）
- 预估剩余时间
- 帧计数器

#### 6.2 交互引导
- 欢迎界面（4种启动方式）
- 分步问询（不用默认值）
- Quick Actions（按钮快捷操作）
- 对话式引导

#### 6.3 视觉设计
- 深色主题（zinc 色系）
- 渐变色按钮（violet → blue）
- 动效反馈（Framer Motion）
- 响应式布局

## 🏗️ 技术架构

### 对话智能层 + 底层引擎层

```
对话智能层（可进化）         底层引擎层（稳定）
├─ 意图识别                  ├─ generateStoryboard()
├─ 流程编排                  ├─ fillStoryboardImages()
├─ 上下文理解                ├─ parseModificationIntent()
├─ 违禁词过滤                ├─ extractKeyFrames()
└─ 与用户共创                └─ 图片/视频生成 API
```

### 核心原则
- ✅ 对话层灵活进化，可通过对话改变生成逻辑
- ✅ 底层引擎标准化，保护核心能力
- ✅ 所有修改由系统智能判断，不依赖显式命令
- ✅ 上下文关联，记住对话历史

---

## 📦 技术栈

- **前端**：Next.js 16 + React + TypeScript
- **AI**：Claude Opus 4.6（PPIO 代理）
- **图片生成**：即梦 Dreamina CLI
- **视频生成**：即梦 Seedance / 可灵 API
- **视频处理**：FFmpeg（关键帧提取/拼接/转换）
- **UI**：Tailwind CSS + Framer Motion
- **数据库**：Prisma + LibSQL（可选）

## 🚀 使用场景

### 故事剧情类
```
1. 上传主角照片 → 风格转换
2. 生成第1段分镜 → 生成视频
3. 提取关键帧（保留角色外观）
4. 作为参考生成第2段分镜 → 角色一致 ✨
```

### 产品宣传类
```
1. 上传产品图 → 深度分析特征
2. 生成分镜（注入产品约束）
3. 批量选择最佳帧
4. 生成视频（只用满意的帧）
```

### 创意短视频
```
1. 描述选题 → 生成3个方案
2. 选择最佳方案 → 生成分镜
3. 对话修改："第3帧更亮一些"
4. 一键生成视频
```
