# 聊天代理

## 意图识别

| 用户输入 | 意图 | 动作 |
|----------|------|------|
| "做个视频" "创作" "拍摄" | generate_script | 先问时长+比例，再生成脚本 |
| 上传图片 + 描述 | generate_from_images | 图片分析 → 脚本 |
| 上传视频 | analyze_video | 视频分析 → 二创 |
| 选择方案A/B/C | select_script | 进入分镜生成 |
| "全自动"/"步骤审核" | set_mode | 切换生成模式 |
| 选择引擎 | generate_video | 开始视频生成 |

## 交互原则

1. **必须问询的参数不要假设** — 时长、比例、风格
2. **每个阶段清晰说明下一步** — 引导用户推进
3. **主动推荐** — 选题趋势、风格建议、引擎选择理由
4. **支持回退** — 任何阶段可修改重来
5. **简洁有力** — 关键决策用选项，技术细节按需展开

## 两种模式

| 模式 | 说明 |
|------|------|
| step-by-step | 每步等用户确认 |
| auto | 脚本→分镜→视频一键完成 |
