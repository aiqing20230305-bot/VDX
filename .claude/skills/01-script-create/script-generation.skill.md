# 脚本生成 Skill

## 流程

```
1. 收集参数    →  选题、时长、比例、风格（缺失则问询）
2. 调用引擎    →  POST /api/script
3. 展示脚本    →  ScriptCard 组件展示，用户选择
4. 进入分镜    →  选定脚本 → 分镜生成
```

## 必要参数（缺一必问）

| 参数 | 说明 | 默认值 |
|------|------|--------|
| topic | 选题描述或上传图片 | 必填 |
| duration | 视频时长（秒） | 必须问询，不要假设 |
| aspectRatio | 画面比例 | 必须问询，不要假设 |
| count | 生成方案数量 | 3 |
| style | 风格倾向 | 不限 |

## 时长选项

- 15秒（短视频）
- 30秒（标准短视频）
- 60秒（中视频）
- 180秒（3分钟）
- 300秒（5分钟，最长）

## 比例选项

| 比例 | 平台 |
|------|------|
| 9:16 | 抖音/快手/TikTok |
| 16:9 | B站/YouTube/横屏 |
| 1:1 | 小红书/Instagram |
| 4:3 | 传统视频 |

## 场景数量规则

- 每个场景 3~5 秒
- 15秒 → 4~5 场景
- 30秒 → 8~10 场景
- 60秒 → 15~20 场景
- 300秒 → 60~80 场景

## 关键代码

- 引擎: `src/lib/ai/script-engine.ts`
- API: `src/app/api/script/route.ts`
