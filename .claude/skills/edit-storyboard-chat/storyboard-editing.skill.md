# 对话式分镜编辑 Skill

**触发场景**：生成分镜后，用户通过自然语言修改提示词

## 核心功能

### 1. 智能意图识别
- 自动检测修改关键词（修改、改、换、调整、更亮、镜头、色调等）
- 解析目标帧（单帧、多帧、全局）
- 提取修改维度（光线、色调、镜头、情绪、细节、构图）

### 2. 自然语言修改指令

#### 单帧修改
```
"让第3帧更亮一些"
"把第5帧改成特写镜头"
"第1帧的光线太暗，加强一下"
```

#### 多帧修改
```
"把第1帧和第5帧改成特写"
"让第2、4、6帧都加上暖色调"
```

#### 全局修改
```
"所有帧都加上电影感"
"给所有画面增加戏剧性光影"
"全部帧都用暖色调"
```

### 3. 支持的修改维度

| 维度 | 关键词示例 | 效果 |
|------|----------|------|
| **光线** | 更亮、更暗、强光、柔光、戏剧性光影 | 调整曝光和光线效果 |
| **色调** | 暖色调、冷色调、复古、赛博朋克 | 改变色彩风格 |
| **镜头** | 特写、远景、俯拍、仰拍、广角 | 调整镜头角度和景别 |
| **情绪** | 忧郁、欢快、紧张、温馨 | 改变画面情绪氛围 |
| **细节** | 增加细节、简化、模糊背景 | 调整画面精细度 |
| **构图** | 居中、三分法、对角线 | 改变构图方式 |

## 使用流程

1. **生成分镜** → 2. **对话修改** → 3. **一键重新生成**

```
用户：生成一个30秒的手机广告视频
助手：[生成9帧分镜]

用户：让第3帧更亮一些，加上电影感
助手：已应用修改：增加亮度 + 电影质感（第3帧）
      修改内容：brighter lighting, cinematic color grading
      [显示"重新生成修改的帧"按钮]

用户：[点击按钮]
助手：[重新生成第3帧]
```

## 实现细节

### API 路由
- `POST /api/storyboard/modify` - 解析修改意图，更新提示词
- `POST /api/storyboard/regenerate-frame` - 重新生成单帧

### 核心引擎
- `prompt-modifier.ts` - 对话式提示词修改引擎
  - `parseModificationIntent()` - 解析修改意图
  - `applyModification()` - 应用修改到帧
  - `applyGlobalModification()` - 全局修改

### 提示词注入策略
- **追加式**：保留原提示词，在末尾追加修改（`original, modification`）
- **智能优化**（未来）：Claude 重新优化完整提示词

## 优势

1. **降低学习成本** - 不需要懂提示词工程，用自然语言即可
2. **快速迭代** - 实时修改，立即重新生成
3. **精准控制** - 可单帧、多帧或全局修改
4. **保留原意** - 追加式修改，不破坏原有风格
