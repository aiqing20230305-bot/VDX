## 任务完成情况

### ✅ Task #169: 优化 ChatMessage 消息分组显示

**问题**:
- 所有消息间距统一（mt-3/mt-4），导致相同角色连续消息视觉上割裂
- 缺乏对话回合的视觉分组感
- 参考 Discord/Slack/Telegram，缺少消息分组优化

**解决方案**:

#### 1. 智能消息分组逻辑
```typescript
// 检查是否与前一条消息来自同一角色
const prevMsg = i > 0 ? messages[i - 1] : null
const isGrouped = prevMsg && prevMsg.role === msg.role && prevMsg.type !== 'question'
const isFirstInGroup = !isGrouped
const isLastInGroup = i === messages.length - 1 || messages[i + 1]?.role !== msg.role
```

**分组规则**:
- 相同角色 + 非问题类型 = 分组
- 问题类型消息始终独立显示（不参与分组）

#### 2. 动态间距
```typescript
className={`... ${isGrouped ? 'mt-1' : 'mt-3 sm:mt-4'}`}
```
- **分组内消息**: `mt-1` (紧凑间距)
- **新对话回合**: `mt-3 sm:mt-4` (正常间距)

#### 3. 渐进式圆角设计

**用户消息** (右侧，青色):
- 单独消息: `rounded-2xl rounded-tr-sm` (顶右角削减)
- 分组首条: `rounded-2xl rounded-br-md rounded-tr-sm` (底右角微圆)
- 分组中间: `rounded-2xl rounded-tr-md rounded-br-md` (顶右+底右微圆)
- 分组末条: `rounded-2xl rounded-tr-md` (顶右角微圆)

**助手消息** (左侧，灰色):
- 单独消息: `rounded-2xl rounded-tl-sm` (顶左角削减)
- 分组首条: `rounded-2xl rounded-bl-md rounded-tl-sm` (底左角微圆)
- 分组中间: `rounded-2xl rounded-tl-md rounded-bl-md` (顶左+底左微圆)
- 分组末条: `rounded-2xl rounded-tl-md` (顶左角微圆)

**设计原理**:
- 保持 Industrial Minimalism 风格（削角设计）
- 通过微圆角（rounded-md）形成视觉连续性
- 避免使用气泡尾巴等复杂装饰，保持简洁

#### 4. 视觉效果对比

**Before (无分组)**:
